# E-Commerce Sales Performance Dashboard — Power BI
## Overview
An interactive two-page Power BI dashboard analysing 10,000+ retail orders across product categories, regions, and customer segments. Built to practice end-to-end dashboard development — from raw data to business insights.

## Dataset

Source: Kaggle — Retail Orders Dataset

https://www.kaggle.com/datasets/ankitbansal06/retail-orders

~10,000 orders with fields including category, region, segment, list price, cost price, quantity, and order date


## Dashboard Pages
### Page 1 — Orders Overview

Total Orders, Units Sold, and Average Discount % KPI cards
Orders by Category (bar chart)
Orders by Region (donut chart)
Monthly Order Trend (line chart)

### Page 2 — Profit & Segment Analysis

Profit by Category (bar chart)
Orders by Segment (donut chart)
Profit by Region (bar chart)
Interactive Category slicer to filter across all visuals

## Key Insights

Office Supplies leads in order volume but Technology generates the highest profit margin
West region dominates both in orders and profitability
Consumer segment accounts for over 50% of all orders
Order volume dips sharply in September, suggesting seasonal patterns worth investigating

## Tools Used

Power BI Desktop
DAX (calculated Profit measure)
